# PesaFit

This is a finance, expense tracker and budgeting application.

## Tech Stack 
The tech stack includes:

### Frontend
 - React
 - Vite
 - TailwindCSS


### Backend
 - Node.js
 - Express
 - MongoDB

 ## Setup and Install
 Clone the repository access frontend directory and install node modules.
 ```
  $ cd frontend
  $ npm install
 ```
## Run React App
To start react application use:
```
 $ npm run dev
```
